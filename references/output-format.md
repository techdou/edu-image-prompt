# Output Format

For each knowledge point, output the following structure:

### [Point Title] | [Visual Type]
- **Recommended Reason**: [Why this visual type works for this specific point]
- **On-Image Copy**:
  - 主标题：[Main Title, <= 12 chars]
  - 副标题：[Short Explanation, <= 24 chars]
  - 重点项：[Key Point 1] / [Key Point 2] / ... (Adapt quantity to visual type, keep each <= 8 chars)
  - 金句：[Memory Line/Gold Sentence]
- **The Prompt**: [A pure, complete, and ready-to-use image-generation prompt including the aspect ratio (default 16:9 unless specified otherwise) and negative constraints]
- **Variants**: 
  - (1) Text-reduced version (Focus on visual metaphor)
  - (2) Pure background version (No text, for manual editing)
  - (3) Icon-centric version (Minimalist style)

# Edu Image Prompt 🎨

> 从知识点生成学生友好的教学配图提示词，服务于 AI 素养课、PPT、公众号、讲义等

## 功能特性

本 skill 将知识点转化为适合教学场景的图像生成提示词（prompt）：

- 面向 **AI 素养课**、PPT 幻灯片、公众号推文、课堂讲义、直播课件、知识卡片等场景
- 生成学生友好（student-friendly）的配图提示词
- 注重教育适配性与视觉传达清晰度

## 使用场景

当需要为教学/科普内容生成配图提示词时使用本 skill —— 例如为某个知识点设计配图、为课件生成插画描述、为公众号文章生成封面提示词等。

## 使用方法

提供知识点或教学内容描述，skill 会生成对应的图像提示词，可直接粘贴到文生图工具（如 Gemini、Midjourney 等）使用。

完整指令见 [`SKILL.md`](./SKILL.md) 与 [`references/`](./references/)。
